Introduction

Le COVID-19 s'agit d'une maladie virale due au virus du coronavirus 2 (SRAS-CoV-2) du syndrome respiratoire aigu sévère. Environ 2,2 millions de personnes ont été infectées et plus de 0,15 million de décès dans le monde, malgré de gros efforts, il 'existe aucun traitement contre cette maladie. Cependant, la prévention et la gestion sont les meilleures options. Dans ce cas nous avons créé une carte interactive de Covid 19 au Maroc qui s’appuie sur ArcGIS Online et incorporé dans un site web créé avec des bases de développement Web.

Méthodologie

  • ArcGIS online

    C'est une solution complète de cartographie et d'analyse. Vous pouvez l'utiliser seul ou étendre votre travail à l'aide d'autres produits ArcGIS. Le travail que vous effectuez peut-être partager et intégré dans ArcGIS.

  • HTML & CSS

    HTML (Hypertext Markup Language) et CSS (Cascading Style Sheets) sont deux des technologies de base pour la construction de pages Web. HTML fournit la structure de la page, CSS la mise en page (visuelle et sonore), pour une variété de dispositifs.

  • JavaScript

    JavaScript est un langage de programmation qui permet d’implémenter des mécanismes complexes sur une page web,qui permet de créer du contenu mis à jour de façon dynamique, de contrôler le contenu multimédia, d’animer des images.
